13 Welcome to the METplus GitHub repository hosted to the community through
14 the Developmental Testbed Center (DTC).
15 
16 METplus is a Python scripting infrastructure around the MET verification tools
17 (and eventually METViewer, a tool used for plotting MET output verification statistics).
18 
19 Documentation for both Users and Contributors can be found here: https://dtcenter.github.io/METplus.
20 Fore more information about the entire suite of tools, please visit the METplus website
21 (https://dtcenter.org/community-code/metplus) and direct questions
22 to met_help@ucar.edu.
23 
24 This infrastructure utilizes the NCEP produtil package, which is a platform-independent
25 weather and ocean forecasting utility package developed at the National Oceanic
26 and Atmospheric Administration (NOAA).
