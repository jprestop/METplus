Returns the cmd with the verbosity option inserted and set after the first space found in the cmd string or after the cmd string if there are no spaces.

There is NO CHECKING to see if the verbosity is already inserted in the command. If cmd is None, None is returned.

Args:

Parameters
cmdOne string, The cmd string to insert the -v option.
Returns
cmd: The cmd string w/ -v <level:1-5> inserted after the first white space or end if no spaces. If cmd is None, None is returned.

The command cmd is a string which is converted to a produtil exe Runner object and than run.

Output of the command may also be redirected to either METplus log, MET log, or TTY.

Some subclasses of CommandBuilder ie. series_by_init_wrapper, run non MET commands ie. convert, in addition to MET binary commands, ie. regrid_data_plane.

Args:

Parameters
cmdA string, Command used in the produtil exe Runner object.
envDefault None, environment for run to pass in, uses os.environ if not set.
ismetcmdDefault True, Will direct output to METplus log, Metlog , or TTY. Set to False and use the other keywords as needed.
app_nameUsed only when ismetcmd=True, The name of the exectable being run.
run_inshellUsed only when ismetcmd=False, will Create a runner object with the cmd being run through a shell, exe('sh')['-c', cmd] This is required by commands, such as ncdump that are redirecting output to a file, and other commands such as the convert command when creating animated gifs.
log_theoutputUsed only when ismetcmd=False, will redirect the stderr and stdout to a the METplus log file or tty. DO Not set to True if the command is redirecting output to a file.
kwargsOther options sent to the produtil Run constructor
